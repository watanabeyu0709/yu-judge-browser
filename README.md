# 这是一个简单的自用babel制作npm包的脚手架

### 安装步骤

    1.请确保全局安装了 babel npm

    2.下载克隆此工程

    3.npm install

    4.npm run build (编译脚本)

### 目录结构

> `lib 发布npm包的环境`

> `src 源文件脚本存放目录`

> .gitignore git 忽略文件

> index.js npm官方入口,可以在package.json中配置

> .babelrc babel配置文件

> package.json 包信息

> README.md 说明文档（本文件）
